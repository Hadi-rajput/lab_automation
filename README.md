# Lab Automation

**Lab Automation** is a project developed to simplify and automate laboratory management tasks.  
It helps in organizing lab experiments, managing resources, and automating repetitive tasks efficiently.  

## Project Overview
- The project is designed to manage laboratory operations in a systematic way.
- All functionalities like adding lab data, updating records, and tracking experiments are automated using backend logic.
- This project was **developed solely by Abdul Hadi**, showcasing independent full-stack development skills.

## Backend Logic
- **PHP** is used to handle all server-side operations, including form submissions, data validation, and CRUD operations.
- **MySQL** database is used to store and manage all lab-related data securely.
- The backend logic includes:
  - Adding new lab records
  - Updating and deleting existing records
  - Validating user inputs to prevent errors
  - Organizing data efficiently for easy retrieval and display

## Features
- Automated lab data management
- Secure and structured database storage
- Efficient backend logic for smooth operation
- Standalone project built independently
